# Blackjack

**Install and compile game using GCC C++ compiler (g++)**

```bash
git clone https://github.com/kgoncharova/cpp-games.git
cd cpp-games/07-blackjack
g++ -o game -I ./lib *.cpp lib/*.cpp
./game
```
